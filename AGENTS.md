# AGENTS.md — skipi-verify-emitter

Этот репозиторий — граница доверия merge-гейта Broker.

- **Агентам-исполнителям (EXEC) писать сюда ЗАПРЕЩЕНО** — by design условия 4:
  fine-grained EXEC-PAT не включает этот репозиторий; наличие write-доступа
  у исполнителя = сломанное условие 4, останов и доклад.
- Изменения (workflow sweeper'а, бампы пина skipi-verify) — только руки
  владельца, либо менеджер skipi-ops по явному слову владельца.
- Ключ App — только environment secret в окружении `emitter`; в код,
  repository secrets или файлы не класть НИКОГДА. Ключ не должен проходить
  через сессию любого агента; компрометация лечится только ротацией ключа.
- PR-workflow, объявляющие `environment: emitter`, блокируются branch policy;
  появление workflow, читающего секрет вне main, — инцидент.
